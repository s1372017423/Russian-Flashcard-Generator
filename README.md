🇷🇺 Russian-Flashcard-Generator

    基于 Python + Obsidian 的自动化俄语背诵方案

🌟 项目简介

还在手动往 Obsidian 里一张张敲单词卡片吗？本项目通过 Python 自动化脚本，将 CSV 格式的俄语词库一键转换为符合 Spaced Repetition 插件语法的 Markdown 笔记。

核心流程：
Excel 录入词汇 -> Python 批量转换 -> Obsidian 插件识别 -> 手机/电脑同步复习
🛠️ 环境准备

在开始之前，请确保你已拥有：

    Python 3.x：用于运行转换脚本。

    Obsidian：并安装了以下插件：

        Spaced Repetition (核心：实现闪卡复习功能)

        Remotely Save (可选：用于多端同步，建议配合坚果云使用)

🚀 详细操作步骤
第一步：数据准备 (CSV)

创建 words.csv 文件，并严格遵守以下规范：

    文件表头：第一行必须为 单词,中文翻译。

    保存编码：必须选择 带有 BOM 的 UTF-8 编码（在 Excel 另存为时选择 CSV UTF-8 (逗号分隔)），否则俄语字母会变成乱码。

示例数据：
| 单词 | 中文翻译 |
| :--- | :--- |
| привет | 你好 |
| спасибо | 谢谢 |
第二步：运行 Python 转换脚本

    将 generate_words.py 和 words.csv 放在同一个文件夹下。

    直接用鼠标点击文件夹窗口上方显示路径的地方（那一长串地址），把它删掉，直接打 cmd 三个字母，按回车。

    之后输入python generate_words.py

    之后按回车

    结果：脚本会自动生成一个名为 俄语单词库 的文件夹，其中每个单词都是一个独立的 .md 笔记。

第三步：Obsidian 插件配置

    将生成的 俄语单词库 文件夹移动到你的 Obsidian 库（Vault）中。

    进入 Obsidian 设置 -> Spaced Repetition：

        在 Flashcard tags 中添加 #flashcards/俄语。

    此时，点击 Obsidian 侧边栏的闪卡图标，你就能看到待复习的俄语单词了！

第四步：移动端同步 (以小米手机+坚果云为例)

    在电脑和手机上同时配置 Remotely Save 插件的 WebDAV（坚果云）。

    点击同步按钮，将词库上传至云端并下载到手机。

⚠️ 常见问题排查 (Troubleshooting)
1. 同步时提示 "503 Service Unavailable"

    现象：点击同步后报错，无法上传文件。

    原因：坚果云等服务器有“频率限制”。由于脚本一次性生成了数百个小文件，频繁的请求会被服务器判定为异常。

    对策：

        静置法：等待几分钟，让文件上传一下就行。

        分批法：初次导入时，可以先放 100 个文件，同步成功后再放入剩下的部分。

2. 同步中断：提示 "超过 50% 的变更"

    现象：提示 20/22 = 90.9% 的文件会被修改或删除 并中止。

    原因：这是 Remotely Save 的安全保护机制，防止误删整个仓库。

    对策：

        进入 设置 -> Remotely Save -> 找到 大规模变更保护 (Large Number of Changes Protection)。

        将百分比临时改为 100%。

        同步完成后，建议调回 50% 以保护数据安全。

3. 弹出了同步成功，但找不到文件夹
   
   愿因把是Obsidian 手机端的“库路径”与云端存放路径不一致
   
   只需要确认你当前打开的仓库文件夹名称，是否和你电脑上同步的那个文件夹名称完全一致即可，如果不一致，那么久更改一下仓库名称就可以了
